**COFFE_PAGE** é um site moderno e totalmente **responsivo** de café, desenvolvido para apresentar produtos, informações e imagens de forma elegante e prática em computadores, tablets e celulares.  

---

## 🌟 Funcionalidades

- Layout **100% responsivo** para desktop, tablet e mobile.  
- Seções de **Home**, **Sobre**, **Menu** e **Depoimentos**.  
- Galeria de imagens para ilustrar produtos.  
- Design limpo e **fácil de navegar**.  

---

## 🛠 Tecnologias Utilizadas

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

- **HTML5** – Estrutura semântica do site.  
- **CSS3** – Estilização moderna, responsiva com **Flexbox e Grid**.  
- **JavaScript (opcional)** – Preparado para interações dinâmicas futuras.  

---

## 📱 Responsividade

O site foi desenvolvido para uma **experiência impecável em qualquer dispositivo**:  

| Dispositivo | Layout |
|-------------|--------|
| Desktop     | Layout completo com todas as seções visíveis |
| Tablet      | Ajuste automático de colunas e imagens |
| Celular     | Navegação simplificada, imagens adaptadas e menus responsivos |
